# DevFinance - controle de finanças Pessoais

Aplicação desenvolvida para gerenciar entradas e saídas financeiras, focada em performance e experiência do usúario (UX).

## Tecnologias
Este projeto foi construído utilizando o "Pilar da Web":
-**HTML5** (Semântica e Acessibilidade)
-**CSS3** (Variáveis, Flexbox/Grid e Responsividade)
-**Javascript Vanilla** (Manipulação de DOM, LocalStorage e Array Methods)

## Desafios Técnicos Superados
-**Precisão Matemática:** Implementação de cálculos utilizando números inteiros(centavos) para evitar erros de ponto flutuante comuns no JS.
-**Persistência Local:** Uso do `LocalStorage` para manter os dados mesmo após o fechamento do navegador.
-**Clean Code:** Organização do código em objetos (`Transaction`, `DOM`, `App`) para facilitar a manutenção.
